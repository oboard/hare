# oboard/moonbit-repl

A REPL for MoonBit

## Quick Start

```bash
# Native
moon run src/main --target native
```

```bash
# Node.js
moon run src/main --target js
```

## Build

```bash
# Native
moon build --target native --release
```

```bash
# Node.js
moon build --target js --release
```
